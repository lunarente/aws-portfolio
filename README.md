# AWS Portfolio — Cloud + Data (Hands-on)

Este repositório reúne projetos práticos em AWS, organizados por trilhas e níveis de maturidade.
O foco é demonstrar tomada de decisão, boas práticas de segurança/custos e entregáveis reproduzíveis.

## Estrutura
- `01_foundations/` — Fundamentos (custos, IAM, storage, rede)
- `02_core-builds/` — Construções essenciais (compute, serverless, APIs)
- `03_data-engineering/` — Pipelines, lakehouse, streaming, catálogo
- `04_iac-devops/` — Infra como código, automação, CI/CD
- `00_meta/` — Padrões, roadmap e critérios de qualidade

## Como ler os labs
Cada pasta de lab possui:
- Contexto e objetivo
- Arquitetura e decisões
- Passo a passo (alto nível)
- Evidências (prints, outputs)
- Lições aprendidas
